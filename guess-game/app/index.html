Guess Game

O Guess Game é um jogo de adivinhação de números. O objetivo é adivinhar qual número o computador está pensando.

Você pode escolher o nível de dificuldade do jogo, que vai determinar a quantidade de chances que você terá.

Você pode ver os resultados dos jogos anteriores no menu "Resultados".
